# 🏥 Previsão de Demanda de Leitos

> Prevê a ocupação de leitos hospitalares e UTI com IA, antecipando picos de demanda (surtos, sazonalidade) para dimensionar recursos e evitar colapso no atendimento.

**🌐 Demo ao vivo (web app interativo):** https://jhondados.github.io/saude-previsao-demanda-leitos/

[![Setor](https://img.shields.io/badge/Setor-Saúde-blue)]() [![IA](https://img.shields.io/badge/IA-Time%20Series-green)]() [![Demo](https://img.shields.io/badge/Demo-Interativa-orange)]() [![Dados](https://img.shields.io/badge/Dados-CSV%2FExcel-success)]()

## 📊 Resultados-chave (KPIs)
**5.3%** MAPE · **otimizada** Ocupação · **14 dias** Antecip. · **-29%** Filas

## 🧠 Tecnologias de IA
Time Series · Machine Learning · Forecasting

## ✨ O que faz
- Previsão de ocupação por ala
- Antecipação de picos
- Dimensionamento de equipe
- Alerta de saturação
- Otimização de recursos

## 🗄️ Fonte de dados real
[DataSUS + ANS](https://datasus.saude.gov.br) — bases de entrada e resultados disponíveis para download em CSV/Excel no app.
